# Control Structures

The technique-level view of control loops: controller, controlled
process, actuators, sensors, reference model, feedback. Used in both
STAMP/STPA and in architectural remedies.

- `overview.md` — the universal control pattern and what this technique adds
- `diagnostic-questions.md` — the five questions used to interrogate any control structure (Q5 applies to social systems only)
- `dangerous-patterns.md` — the four recurring patterns behind the worst failures (Pattern 4 applies to social systems only)
- `technical-examples/` — engineered-system illustrations

For the social-systems extension see `../justification-rungs/`.

`knowledge/system-catalogues/social-systems/cross-system/control-structure-profiles.md`
(per-system profiles) and
`knowledge/system-catalogues/social-systems/cross-system/remedies-case-studies.md`
(worked remedies).
