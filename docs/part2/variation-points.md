# Variation Points in Social Systems

## Six Primary Variation Points

### VP1: Source of Authority

| Variant | Used By |
|---------|---------|
| Divine mandate | Theocracy, traditional monarchy |
| Hereditary right | Kingdom |
| Popular sovereignty | Republic, Verein |
| Ideological mandate | One-party state |
| Capital ownership | Corporation |
| Scholarly merit | University |
| Command hierarchy | Military |
| Biological bond | Family |

### VP2: Membership Boundary

| Variant | Used By |
|---------|---------|
| Birth (citizenship) | Kingdom, Republic, Theocracy |
| Voluntary joining | Verein, Church, Corporation |
| Competitive selection | University, Military |
| Biological kinship | Family |

### VP3: Decision-Making Mechanism

| Variant | Used By |
|---------|---------|
| Autocratic decree | Kingdom, One-party state |
| Democratic vote | Republic, Verein |
| Board vote (capital-weighted) | Corporation |
| Collegial deliberation | University |
| Command order | Military |
| Consensual negotiation | Family |
| Doctrinal interpretation | Theocracy, Church |

### VP4: Succession Mechanism

| Variant | Used By |
|---------|---------|
| Hereditary | Kingdom, Family |
| Periodic election | Republic, Verein |
| Board appointment | Corporation, University |
| Internal promotion | Military, One-party state |
| Clerical selection | Church |

### VP5: Legitimation Narrative

Each system tells a different story about why its authority is valid — from tradition (Kingdom) to market success (Corporation) to love (Family).

### VP6: Norm Enforcement Mechanism

Ranges from state coercion (Republic, Kingdom) to social pressure (Verein, Family) to ecclesiastical discipline (Church) to military justice (Military).

## Constraint Rules

Not all combinations are compatible. VP1 cascades most strongly: if VP1 = "hereditary right," then VP4 cannot be "periodic election" for the head of state. Understanding these constraints is what makes product line thinking useful for institutional reform — it tells you which changes are structurally compatible before you attempt them.
