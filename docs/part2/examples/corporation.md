# Example: Corporation (For-Profit)

The for-profit corporation is the organizational family's most studied variant. Its SE decomposition reveals a system optimized for a single metric (profit) with governance designed to serve the capital-holders — a binding that produces both extraordinary efficiency and well-known pathologies.

[**→ Interactive D3 Visualization**](../../interactive/corporation.html){ .md-button }

## SE Decomposition

The full five-level decomposition is shown in the interactive visualization linked above. Click any node to see its description, parent links, and child links. Use the Table view for the complete traceability matrix.

## Variation Point Bindings

VP1 = capital ownership (shareholders), VP2 = employment contract (voluntary, conditional), VP3 = board vote weighted by shares, VP4 = board appointment. The key structural feature: employees have no formal authority despite being the system's primary operators.
