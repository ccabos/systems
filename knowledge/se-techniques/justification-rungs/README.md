# Justificatory Rungs

A ladder of standards for what counts as a *decisive reason* to
issue a control action or accept a corrective signal. Rungs run from
0 (raw coercion) to 6 (deliberative legitimacy). Each later rung
adds an internal or external check that earlier rungs lack.

The technique is an extension to STPA and to the control-structures
diagnostic for **social systems**. In engineering systems, all
arrows usually operate at rung 3-4, so the technique is uninteresting
there. In social systems, the rung of each arrow varies, the rungs
on either end of a loop often differ, and that mismatch is itself
one of the dominant failure modes.

## Tradeoff to keep visible

Leveson's STPA is deliberately neutral about which controllers are
"better." The rung ladder is openly normative — rung 6 sits above
rung 0 in a way that rung 0 cannot reciprocate. Use this technique
where epistemic standards matter (social, institutional, policy
systems). Do not import it into engineered safety analyses where it
adds no information.
