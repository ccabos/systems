# One-Party State — SE Decomposition

Extracted from `projects/systems-introduction-book/docs/part2/ten-systems.md`
§2.1.4. IDs use the `O*` prefix as in the source.

## Goals (OG)

- **OG1** — Implement the party's ideology as the organizing principle of society
- **OG2** — Maintain the party's monopoly on political power
- **OG3** — Ensure order, security, and territorial integrity
- **OG4** — Deliver economic development and social services as proof of party legitimacy

## Requirements (OR)

- **OR1** — Constitutional enshrinement of the party's leading role
- **OR2** — Fusion of party and state — parallel hierarchies at every administrative level
- **OR3** — Ideological conformity — censorship, propaganda, suppression of opposition
- **OR4** — Security apparatus loyal to the party, not the state
- **OR5** — Centralized economic planning or state-directed capitalism
- **OR6** — Cadre system — party membership as prerequisite for advancement

## Functions (OF)

- **OF1** — Party-directed legislation — laws originate in party committees, rubber-stamped by assembly
- **OF2** — Cadre management — recruit, evaluate, promote, and discipline officials through the party
- **OF3** — Ideological control — propaganda, censorship, political education
- **OF4** — Security and surveillance — monitor population, suppress dissent
- **OF5** — Economic direction — set plans, allocate investment, manage state enterprises
- **OF6** — Military command under party authority

## Logical Architecture (OL)

- **OL1** — Party-State Legislative Subsystem — politburo/standing committee, people's congress
- **OL2** — Cadre & Personnel Subsystem — organization department, nomenklatura
- **OL3** — Propaganda & Control Subsystem — media, censorship, education ministry
- **OL4** — Security Subsystem — secret police, internal security forces, surveillance
- **OL5** — Economic Planning Subsystem — state planning commission, SOEs, industrial ministries
- **OL6** — Military Subsystem — party-controlled armed forces, political commissars

## Physical Implementation (OP)

- **OP1** — Politburo / Central Committee / Standing Committee of the NPC
- **OP2** — Organization Department, party schools, cadre evaluation files (档案)
- **OP3** — State media (Pravda, Xinhua, CCTV), Great Firewall, propaganda department
- **OP4** — Secret police (Stasi, KGB, MSS), surveillance infrastructure
- **OP5** — State Planning Commission (Gosplan), state-owned enterprises, 5-year plans
- **OP6** — People's Liberation Army, political commissar system
