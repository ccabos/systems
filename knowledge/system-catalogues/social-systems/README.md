Case studies of non-technical systems that are social in character.

## Cross-system material

`cross-system/` contains material that compares multiple members of
the catalogue: the universal functional slots, the comparison matrix,
variation-point bindings, reuse tables, and the social system platform.

## Adding a new system

Copy `_template/` to a new folder named after the system and fill in
each file. See `_template/README.md` for the expected structure and the
questions each file must answer.
